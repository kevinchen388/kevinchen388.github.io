Plopper

--
A responsive website that helps you find the nearest bathroom, developed at MHacks 11.
Tracks users' location and shows the nearest restrooms on a map. Users can then select each restroom to view pictures, reviews, and details. Users can also log in to write reviews and add restrooms to the database.
Utilizes Python, Javascript, HTML/CSS, and Google Maps API.

DJ 183

--
A website that hosts a music quiz engine, created for the EECS 183 Final Project.
Allows the user to select a music genre and then guess what song is playing. DJ 183 then reveals the correct answer and plays the corresponding music video implemented from YouTube. Users also have the choice to save their score on the leaderboard.
Utilizes Python, HTML/CSS, Google Firebase, and Apple Music API.
